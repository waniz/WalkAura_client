Skills changes:
  • Equipped slots get hotkey labels (1-6) and school-tinted glow when filled. Empty slots dashed. Instruction text "Tap to slot, long-press to remove" lives in header so the UX is discoverable.
  • School chips become rounded pills with colored school-dot indicators (not bold text headers). Horizontal scroll for breathing room. Multi-select.
  • Spell cards become 2-col grid with 44px icon + name + cost/cast/CD line + one effect line. School-colored left edge + name color = instant school recognition. Equipped spells get gold ★ corner + gold glow halo.
  • Class tabs (Mage/Paladin/Utility/Blood/Dark/Arcane) become parchment-style tabs above school chips — clearer mental model than 6-pill row.
  • 3-bar HUD now shown — HP/Shield/MP stacked. Each bar gets "HP/SH/MP" prefix.
Talents changes:
  • Class emblem center replaces mystery black ring. Gold-bordered glowing disk with class name + icon. Anchors the whole tree visually.
  • Radial branch structure — 4 branches (Fire/Frost/Arcane/Dark) splay from center. Branch labels at cardinal edges of canvas. Solid colored connectors for invested paths, dashed muted lines for unlearned.
  • Rank chip on every node (e.g. "2/3" or "0/3"). Maxed nodes get filled gold chip; partial gets gold-outline chip; locked nodes greyed.
  • Node states: locked (40% opacity + grayscale), available (full color border), partial (school-colored glow), maxed (gold rim + warm radial bg + gold rank chip).
  • Header panel tightened — Spent/Available stats left, +1/+5/RESPEC buttons right. Loud red `[- 0 +]` cluster from current screen replaced with neutral +1/+5 quick-spend buttons.
