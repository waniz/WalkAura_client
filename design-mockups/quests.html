Quest UI structure:
  • List view (left): 4 filter chips — Active / Available / Daily / Done. Counts inline. Pip on Daily when something new dropped. Daily reset timer banner up top when relevant.
  • Quest cards: Tier-color left edge (matches rift/location tier system). Story quests get gold edge. "TRACKED" flag in corner = appears in location tracker. Cards differ by type: kill/gather use multi-objective checklist, single-objective uses one progress bar.
  • READY-to-turn-in state: Card lights up with gold glow + "READY" pip in corner. Players can see at a glance which quests need to be handed in.
  • Reward chips: XP/gold in gold, items in their quality color (rare blue, epic purple). Matches inventory quality language — same color = same meaning everywhere.
  • Detail view (right): Painted illustration anchors mood, italic lore text overlays it. Giver block tells you who/where. Objectives in dark cards with per-objective progress. Rewards in clear grid. Track/Abandon actions at bottom.
  • Empty states: "No active quests — visit Northwatch to pick one up" + travel button to the quest hub.
