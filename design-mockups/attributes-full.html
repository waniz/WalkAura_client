Attributes sub-tabs changes:
  • Sub-tab bar becomes painted segmented control. Active tab fills with sub-tab's accent color (Offensive=red, Defensive=blue, Steps=gold, Sustain=green). Each tab has an icon glyph so it scans visually.
  • Summary bar below sub-tabs shows the 3 most relevant high-level stats for that view (Phys ATK / Magic ATK / Crit on offensive, etc.). Gives a glance answer before you scroll into rows.
  • Stat rows get section headers ("Combat Ratings" / "Damage Amplifiers" / "Mitigation" / "Resistances"), 2px sub-tab-color left edge for rhythm, 22px icon column, name, value, percent. Zero values dimmed to 55% opacity so non-zero stats jump out. Hi values (notable) get success-green value text.
  • Damage Amplifiers / Resistances use element colors as left-edge (fire red, frost blue, dark purple, blood crimson, etc.) — matches school colors used in skills + spells.
  • Top stat strip compresses primary stats (STR/AGI/VIT/INT/SPI/LCK) to 3x2 grid; visible on every sub-tab so they stay context.
